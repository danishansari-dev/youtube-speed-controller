Privacy Policy

Last Updated: May 2026

Universal Video Speed Controller does not collect, store, transmit, or sell any personal user data.

Data Collection

This extension does not:

  • collect personal information
  • track browsing history
  • store passwords
  • access private messages
  • sell user data
  • use third-party analytics

Local Storage

The extension may store user preferences locally using the browser storage API. This data never leaves the user's device.

Permissions Usage

Permissions are required only for:

  • detecting HTML5 video players
  • controlling playback speed
  • saving extension preferences
  • displaying floating controls

Third-Party Services

This extension does not use external servers, tracking scripts, or third-party analytics.
